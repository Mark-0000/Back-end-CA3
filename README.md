# Back-end-CA3

## Required
XAMPP web server

### how to install
1. clone repository to C:->xampp->htdocs->todoapp\
2. import the todo.sql file on php my admin page\
3. launch xampp control panel\
4. start Apache and MySQL\
5. open http://localhost:port/todoapp\
